# Decision Engine

An interactive behavioral research system designed to study how individuals make decisions under pressure.

Instead of static surveys, this system places users inside a dynamic environment where they must respond to evolving situations in real time. Every action, delay, and adjustment is recorded and translated into measurable behavioral patterns.


## Overview

Decision Engine simulates high-pressure scenarios where users must:

- make time-sensitive decisions  
- balance competing priorities  
- adapt to changing conditions  
- respond to incomplete or uncertain information  

The system captures not just *what* users choose, but *how* they arrive at those decisions.


## Core Idea

Traditional research methods rely heavily on self-reported data.

This project explores an alternative:

> Observing behavior through interaction instead of asking about it.

By embedding users in a controlled decision environment, we can measure:

- reaction speed  
- risk-taking tendencies  
- consistency across similar situations  
- adaptability after consequences  
- decision conflict under uncertainty  


## Features

### Interactive Decision System
- Multi-stage decisions (intent → calibration → commit)
- Real-time system feedback
- Persistent system state across scenarios

### Dynamic Scenarios
- Context-neutral, relatable situations
- Narrative-style micro-scenarios for immersion
- Evolving tension across the session

### Multiple Interaction Modes
- Action selection + risk calibration  
- Interrupt decisions (time-limited)  
- Resource allocation  
- Priority ordering  
- Reversal scenarios  

### Behavioral Metrics
The system captures and analyzes:

- **Average Decision Time**  
- **Risk Preference**  
- **Consistency Score**  
- **Adaptability**  
- **Confidence Gap**  
- **Decision Conflict Score**  
- **Strategy Type (derived)**
